## :mega:프로젝트 소개
Final Project 기간에 제작된 교육 플랫폼 기반의 학교 사이트입니다.<br/>
현실의 해리포터를 좋아하는 매니아층과 세계관 속 호그와트 관계자들을 타겟으로 제작되었습니다.
